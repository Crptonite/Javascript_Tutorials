Contribute to a GitHub Repository

Follow the steps to make your first contribution

Step 1: Fork the Repository

Click the Fork button in the top-right corner of the repository.

Step 2: Clone the Repository

Copy the repository URL and run this command in your terminal:

git clone https://github.com/your-username/repo.git

Step 3: Make Changes

Make your changes in your local repository and commit them:

git commit -m "Your commit message"

Step 4: Push Changes

Push your changes to your GitHub repository:

git push origin branch-name

Step 5: Create a Pull Request

Go to your forked repository and create a pull request with the changes.
